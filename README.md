# Mini Polling Application

A **full-stack mini polling application** where admins can create polls and users can vote. Built with **Node.js, Express,  and a frontend

---

## Features

- **Admin Panel**
  - Create new polls
  - Set poll as active/inactive
  - View poll results

- **User Panel**
  - View all active polls
  - Vote on polls
  - See poll results

- **Full-Stack**
  - Backend: Node.js + Express
  - Frontend: React.js 
  - RESTful APIs for communication between frontend and backend
